# Introduction

Have you ever thought programming was interesting, but don't know where to start? Do you want to automate repetitive tasks, start a new career, and learn what your computer is capable of? Whatever your reasons, this book is your foot in the door to programming in general and the Python programming language in particular.

## Learning Objectives:

Learn the fundamentals of programming, including what a programming language is and how to run programs you create.
Introduce basic programming concepts like variables, if statements, and loops.
Learn to read and explain source code, and predict what a script will do.
Get resources to install Python and where to go from here.

## Prerequisites:

Some highschool level math and basic computer literacy is recommended.
